##Esta es la API del proyecto EnergySystem: Sistema de Monitoreo de consumo Energía en el hogar con tecnología Iot.

Esto debe ir en .env:
```
DB_USER =
DB_PASSWORD=
DB_HOST=
DB_DATABASE=
DB_PORT=
```

##Instalar dependecias 
```
npm install
```

##Ejecucion Ejecutar en consola con 
```
npm run dev
```
